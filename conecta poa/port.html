Meu Portfólio

Este projeto apresenta uma tela de login moderna, criada com HTML5 e CSS. O design utiliza cores vibrantes, gradiente e um card centralizado com bordas arredondadas e sombra suave. A interface é simples e direta, contendo: Campo de e-mail Campo de senha Botão de acesso Layout totalmente responsivo O objetivo do projeto é demonstrar habilidades básicas de interface, organização visual e responsividade.
O segundo projeto é uma página inicial de portfólio, contendo seção sobre mim e área de projetos. Ele utiliza Grid responsivo e cartões animados. Inclui: Cabeçalho com degradê Seção “Sobre Mim” Seção “Projetos” com 3 cards estilizados Rodapé simples com espaço para links O design foi criado com foco em claridade, simplicidade e destaque visual, usando tons fortes e sombras suaves para dar um aspecto profissional.
Este projeto simula um portal de notícias, com blocos de artigos organizados de maneira limpa e profissional. Utiliza cartões com efeito de elevação ao passar o mouse, garantindo boa legibilidade e hierarquia visual. Os principais elementos são: Cabeçalho chamativo com gradiente azul Dois artigos de exemplo com título e descrição Estrutura responsiva para dispositivos móveis Foi desenvolvido para demonstrar organização semântica com
e layout limpo para leitura.
